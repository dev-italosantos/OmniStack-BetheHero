## Objetivo do projeto 

O projeto Be The Hero irá conectar ONG's que necessitam de ajuda com pessoas que queiram ajudar.
Nele, ONG's iram cadastrar projetos e as pessoas interessadas poderam doar um valor qualquer para a causa.
